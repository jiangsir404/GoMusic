# GoMusic
A android muisc player

# 够音乐
一款仿网易云音乐的简易音乐播放器

# 团队
Make by @rivir，@zyl，@lh

# 功能

- [x] 注册
- [x] 登录
- [x] 列表显示
- [x] 播放音乐
- [x] 同步显示歌词
- [ ] 搜索歌曲
- [ ] 调用网络歌曲api,获取更多歌曲
- [ ] 用户中心
- [ ] 注销
